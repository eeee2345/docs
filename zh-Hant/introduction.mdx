Panguard AI 是一個開源安全平台，同時保護 **AI 代理程式**與**端點**。作為 [OpenClaw](https://openclaw.ai) skill 生態系的安全防護層，Panguard 在 skill 到達您的代理程式之前，掃描 prompt injection、tool poisoning 和隱藏威脅。同時為開發者與中小企業帶來企業級端點防護，無需專業安全團隊。

核心理念很簡單：**當有事發生，Panguard 通知您。沒事發生時，您什麼都不用做。**

## 為什麼選擇 Panguard AI？

AI 代理程式正透過 OpenClaw 等 skill 生態系獲得強大能力 -- 但每次安裝 skill 都是一次信任決策。同時，傳統端點安全工具需要深厚專業知識，授權費用高達六位數。

Panguard AI 採取不同的方式：

- **AI skill 安全** -- 安裝前掃描 skill 的 prompt injection、tool poisoning、隱藏 Unicode 和編碼載荷
- **一行指令安裝** -- 無需設定檔、無需調校、無需專業術語
- **白話通知** -- 透過 Telegram、Slack 或 Email 以您能理解的語言接收警報
- **AI 自動判斷嚴重性** -- 系統自動分類、回應與報告，無需人工介入
- **持續進化** -- 行為基線會自適應您的特定環境

## 快速安裝

```bash
curl -fsSL https://get.panguard.ai | bash
```

然後進行驗證並執行首次掃描：

```bash
panguard login
panguard scan --quick
```
