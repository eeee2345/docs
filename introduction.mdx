Panguard AI is an open-source security platform that protects both **AI agents** and **endpoints**. As the security layer for the [OpenClaw](https://openclaw.ai) skill ecosystem, Panguard scans AI skills for prompt injection, tool poisoning, and hidden threats before they reach your agents. It also brings enterprise-grade endpoint protection to developers and small businesses -- without requiring a security team.

The core philosophy is simple: **when something happens, Panguard tells you. When nothing happens, you do nothing.**

## Why Panguard AI?

AI agents are gaining powerful capabilities through skill ecosystems like OpenClaw -- but every installed skill is a trust decision. Meanwhile, traditional endpoint security tools require deep expertise and cost six figures.

Panguard AI takes a different approach:

- **AI skill security** -- scan skills for prompt injection, tool poisoning, hidden Unicode, and encoded payloads before installation
- **One command install** -- no configuration files, no tuning, no jargon
- **Plain language notifications** -- receive alerts via Telegram, Slack, or Email in language you understand
- **AI auto-judges severity** -- the system triages, responds, and reports without human intervention
- **Gets smarter over time** -- behavioral baselines adapt to your specific environment

## Quick install

```bash
curl -fsSL https://get.panguard.ai | bash
```

Then authenticate and run your first scan:

```bash
panguard login
panguard scan --quick
```
